# Scientific-Colormaps-for-Godot
Common colormaps used in matplotlib ported over to Godot for any use case. Provided as a simple editor script to generate Godot Gradient 1D resources

Simply add the script to your project, find it in the FileSystem, right click and select RUN!
<img width="1882" height="754" alt="image" src="https://github.com/user-attachments/assets/df632e29-a584-49b2-a971-fad720d83168" />

The Gradient resources will be added to the root of your project, use them however you like.
